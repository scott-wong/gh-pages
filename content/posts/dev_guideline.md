# 企业架构系列：开发规范

## 版本号
【强制】版本号遵循 [SemVer 2 标准](https://semver.org/)  

**语义化版本 2.0.0**  
版本格式：主版本号.次版本号.修订号，版本号递增规则如下：

- 主版本号：当你做了不兼容的 API 修改，
- 次版本号：当你做了向下兼容的功能性新增，
- 修订号：当你做了向下兼容的问题修正。  

先行版本号及版本编译元数据可以加到“主版本号.次版本号.修订号”的后面，作为延伸。
